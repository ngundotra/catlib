Lean Declaration

axiom freedom_justifies_permission :
  ∀ (p : Person),
    p.hasFreeWill = true →
    -- The value of this person's freedom outweighs the evil
    -- they might freely commit
    True
❅

Why This Axiom?

The freedom-justifies-evil axiom. Freedom is valuable enough to justify God's permitting the evil that free creatures do.
